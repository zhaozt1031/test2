这个仓库会存放，git对我们项目代码代码备份的文件 要备份到哪里，就在当前文件夹点击打开git bash here 初始化仓库：git init 设置用户名：git config --global user.name “zhaozt” 设置邮箱：git config --global user.email “zitong536@163.com” git add ./文件名字 git add ./ 加好多个文件 git commit -m "文件的说明" git commit --all -m "文件的说明" ：这是一次性直接放到门口，如果这么写，不用写add 每次代码更新之后要提交 git status 可以查看当前的状态：nothing to commit 说明未写内容 红色modified 说明写了未添加提交 .gitignore 在这个文件中可以设置要被忽略的文件或者目录，被忽略的文件不会提交到仓库中 /.idea会忽略.idea文件 /js会忽略js目录里所有文件 /js/*.js会忽略js目录下所有js文件 git log 查看历史提交的日志 git log --online 可以看到简洁版的日志 git reset --hard Head~0 回退到第几个 0就是最后一次提交的版本 git reset --hard 版本号 通过版本号切换版本 git reflog 可以查看所有对版本的操作 git branch dev 创建了一个dev分支 git branch 看见所有的分支 git checkout dev 切换到dev分支 git merge dev 切换回master后，合并dev到master gitHub 允许别人通过git上传代码，提交代码到gitHub上（只是个服务器） 登陆网站，建一个仓库，上传代码 git push （github里面的地址） 要传的分支 git push https://github.com/zhaozt1031/test1.git master
